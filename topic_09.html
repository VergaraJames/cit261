Standard JavaScript Events Including those for Mobile Devices ( Ex. onTouchBegin, onLoad, etc.) and Animation and Transition Events

How HTML Events can it be used by developers?

HTML events are "things" that happen to HTML elements.
When JavaScript is used in HTML pages, JavaScript can "react" on these events .

onchange An HTML element has been changed.
onclick The user clicks an HTML element.
onmouseover The user moves the mouse over an HTML element.
onmouseout The user moves the mouse away from an HTML element.
onkeydown The user pushes a keyboard key.
onload The browser has finished loading the page.

Difference between onmousemove, onmouseenter and onmouseover.

The onmousemove event occurs every time the mouse pointer is moved over the div element.

The onmouseenter event only occurs when the mouse pointer enters the div element.

The onmouseover event occurs when the mouse pointer enters the div element, and its child elements (p and span).
