Topic 06

DOM Manipulation Using createElement, appendChild, insertBefore, removeChild, etc.

DOM - Document Object Model was created by the browser when a webpage is loaded. It is a graphical form of elements/nodes.
Developers can use JavaScript to change the DOM or remove HTML elements, or change CSS Styles, read and change element attributes (href, src, alt), create new HTML elements and insert them into the DOM.

Properties and Methods

The following properties and methods can be used on all HTML elements:

Property / Method Description
